🔧 Solusi untuk Masalah Koneksi

  1. Ganti DNS ke Google DNS:
    • Windows: Control Panel → Network and Internet → Network Connections → Properties → IPv4 → Use DNS: 8.8.8.8 dan 8.8.4.4
    • Atau gunakan Cloudflare DNS: 1.1.1.1 dan 1.0.0.1
  2. Matikan Firewall/Antivirus sementara:
    • Windows Defender Firewall → Turn off (temporary)
    • Atau tambahkan exception untuk domain *.supabase.co
  3. Flush DNS Cache:
    • Buka Command Prompt (CMD) as Administrator
    • Jalankan: ipconfig /flushdns
  4. Cek Proxy Settings:
    • Windows Settings → Network & Internet → Proxy
    • Pastikan "Automatically detect settings" ON
    • Atau matikan proxy jika tidak digunakan
  5. Restart Network Adapter:
    • Control Panel → Network Connections
    • Disable lalu Enable kembali network adapter
  6. Cek dengan Browser Lain:
    • Coba buka aplikasi di browser lain (Chrome, Firefox, Edge)
    • Jika berhasil, clear cache browser yang bermasalah
